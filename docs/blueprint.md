# **App Name**: Dockwood Furniture's

## Core Features:

- Dynamic Product Catalog: Showcase all furniture and timber products with dedicated product cards featuring images, names, categories, and quantity status. Includes 'Inquire Price' buttons that link directly to WhatsApp with pre-filled messages.
- Secure Admin Panel: A protected admin interface using Firebase Authentication to manage all aspects of the product inventory, sales, and content. Only authorized administrators can log in.
- Comprehensive Product Management: Admins can add new products, edit existing details, delete items, upload product images to Firebase Storage, assign categories, and update product quantities in the Firestore database.
- Inventory & Sales Tracking: System for administrators to set and update product stock levels, record sales transactions, and automatically decrease quantities in Firestore. Includes an inventory alert for low stock items.
- Advanced SEO Framework: Integration of meta titles, descriptions, OpenGraph metadata, structured Schema.org LocalBusiness data, semantic HTML, SEO-friendly URLs, image optimization, and lazy loading for high search engine ranking.
- Interactive Contact & Location Module: Display essential contact information including phone and WhatsApp links, a user inquiry contact form, and an integrated Google Map for the business location in Bombolulu, Mombasa.
- AI-Powered Product Description Tool: An administrative tool that uses generative AI to assist in creating compelling and SEO-friendly product descriptions based on product name and category, enhancing content creation efficiency.

## Style Guidelines:

- Primary color: Deep, warm brown (#664726) for a professional and natural 'wood' feel, reminiscent of rich timber. This provides a stable and reliable foundation for the visual design.
- Background color: Very light, subtle cream (#FCFAF7) with a warm undertone, enhancing readability and offering a clean, airy canvas that complements the primary brown.
- Accent color: Rustic burnt orange (#BF4F29) to highlight interactive elements and call-to-actions, providing a warm and inviting contrast that draws attention effectively.
- Headline font: 'Poppins' (sans-serif) for a modern, geometric, and contemporary look. Body text font: 'PT Sans' (sans-serif) for legibility across various screen sizes and a friendly yet professional tone.
- Utilize clean, modern line-art icons that align with the minimalist aesthetic, used consistently across navigation, product features, and administrative actions to improve clarity and user experience.
- Adopt a spacious, grid-based layout for product displays, prioritizing high-quality imagery and ensuring content readability. Implement a mobile-first responsive design for seamless viewing on all devices, from desktops to smartphones.
- Subtle and smooth transitions for page loads, product card interactions, and button hover states. These animations are designed to enhance the user experience by adding a refined touch without being obtrusive.
